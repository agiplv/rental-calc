# rental-calc

## Запуск на GitHub Pages

1. Все файлы приложения теперь находятся в корне репозитория (`index.html`, `manifest.json`, `service-worker.js`, иконки и т.д.).
2. В корне репозитория есть файл `.nojekyll` (важно для корректной работы PWA и web components).
3. В настройках репозитория GitHub выберите публикацию Pages из ветки `main` и директории `/` (корень).
4. После публикации приложение будет доступно по адресу: `https://<ваш-логин>.github.io/<repo>/`.
