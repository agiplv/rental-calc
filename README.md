# rental-calc

## Запуск на GitHub Pages

1. Все файлы приложения должны быть в папке `public/` (index.html, manifest.json, service-worker.js, иконки и т.д.).
2. В корне репозитория должен быть файл `.nojekyll` (уже добавлен), чтобы GitHub Pages корректно обслуживал файлы без обработки Jekyll.
3. В настройках репозитория GitHub выберите публикацию Pages из ветки `main` и директории `/public`.
4. После публикации приложение будет доступно по адресу: `https://<ваш-логин>.github.io/<repo>/`.
